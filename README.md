현대식품관 투홈 리뉴얼은 React 기반 E-commerce 팀 프로젝트로 라우팅, 전역 상태, 상품 데이터 구조 등을 모듈화해 유지보수성을 강화했습니다. <br/>
Redux Toolkit으로 장바구니, 좋아요, 인증 상태 등을 관리하고 상품 리스트 기능(정렬, 필터링, 담기)을 전역 상태 기반으로 구현했습니다. <br/>
브랜드소개, 선물하기, 전체 상품 카테고리 페이지를 담당해 디자인부터 개발까지 진행했으며, 슬라이더 제어, 동적 라우팅, 필터, 정렬 로직, 상태 기반 UI 등 핵심 기능을 직접 구현했습니다. <br/>
일부 페이지는 반응형으로 제작해 다양한 디바이스 환경을 지원합니다.
